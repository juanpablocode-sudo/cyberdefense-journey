# 🧮 Sistemas de Ecuaciones

## 🎯 Concepto

Conjunto de ecuaciones con múltiples incógnitas.

## 📌 Ejemplo

x + y = 10  
x - y = 2  

## 💻 Aplicación

- Modelado de sistemas  
- Resolución de problemas complejos  

## 🔐 En ciberseguridad

Base matemática para:
- criptografía  
- algoritmos de seguridad
