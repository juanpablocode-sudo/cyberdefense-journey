# 🚀 Crecimiento Exponencial

## 🎯 Concepto

Una función exponencial crece rápidamente en el tiempo.

## 📌 Ejemplo

f(x) = 2^x

## 💻 Aplicación

- Crecimiento de datos  
- Escalabilidad  

## 🔐 En ciberseguridad

- Propagación de malware  
- Ataques de fuerza bruta  
- Incremento de tráfico malicioso  

## 🧠 Conclusión

El crecimiento exponencial permite entender cómo un ataque puede escalar rápidamente si no se controla.
