# 📉 Límites

## 🎯 Concepto

El límite describe el comportamiento de una función cuando se aproxima a un valor.

## 📌 Ejemplo

Cuando x → 0, la función puede acercarse a un valor específico.

## 💻 Aplicación

- Análisis de comportamiento de sistemas  
- Evaluación de cambios graduales  

## 🔐 En ciberseguridad

Permite analizar cómo evoluciona un sistema ante múltiples intentos de acceso o carga.
