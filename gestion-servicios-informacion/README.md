# 🏢 Gestión de Servicios de Información

Materia orientada a la administración y gestión de servicios IT dentro de organizaciones.

## 🎯 Objetivo

Comprender cómo se diseñan, implementan y gestionan servicios tecnológicos de manera eficiente y segura.

## 📚 Contenidos

- ITIL (Information Technology Infrastructure Library)
- SLA (Service Level Agreement)
- KPI (Indicadores de rendimiento)
- Gestión de incidentes
- Seguridad de la información

## 🔐 Aplicación en Ciberdefensa

- Gestión de incidentes de seguridad
- Respuesta ante ataques
- Continuidad del negocio
- Protección de datos

## 📁 Contenido del repositorio

- Resúmenes teóricos
- Casos prácticos
- Modelos de gestión

## 🚀 En progreso

Se agregarán ejemplos reales de incidentes y análisis de casos.
