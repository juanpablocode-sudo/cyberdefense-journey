# 🌐 Lab: Análisis de Conexiones de Red

## 🎯 Objetivo

Visualizar conexiones activas del sistema.

## 🛠️ Herramientas

- netstat

## 🔍 Comando

netstat -tuln

## 📊 Análisis

- Puertos abiertos
- Servicios activos

## 🔐 Aplicación en seguridad

Permite detectar conexiones sospechosas o servicios vulnerables.
