# 🔐 Lab: Análisis de Permisos en Linux

## 🎯 Objetivo

Identificar archivos con permisos inseguros.

## 🛠️ Herramientas

- find
- chmod

## 🔍 Procedimiento

1. Ejecutar script de análisis
2. Detectar archivos con permisos 777

## ⚠️ Riesgo

Los permisos 777 permiten acceso total a cualquier usuario.

## 🔐 Aplicación en seguridad

- Detección de malas configuraciones
- Prevención de accesos no autorizados
