# 🧪 Lab: Análisis de Procesos en Linux

## 🎯 Objetivo

Identificar procesos activos y detectar posibles anomalías.

## 🛠️ Herramientas utilizadas

- ps
- top

## 🔍 Procedimiento

1. Ejecutar:
   ps aux

2. Analizar:
   - Usuario
   - Consumo de CPU
   - Procesos sospechosos

## 📊 Resultado

Se identificaron múltiples procesos del sistema.

## 🧠 Conclusión

El análisis de procesos permite detectar actividad sospechosa en el sistema.
